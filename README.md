# Snake-Game


# Sobre o projeto

### O Snake game tem como objetivo atingir uma cobra de maior dimensão possível. O jogo chega ao fim quando a cabeça da cobra encostar em qualquer parte de seu corpo ou se o jogador decidir encerrá-lo.
